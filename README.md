# Geometry Calculator (C++)

Простая программа для вычисления площади и периметра геометрических фигур.  
Демонстрирует применение ООП, полиморфизма и RAII в C++.

## Фигуры
- Круг (`Circle`)
- Прямоугольник (`Rectangle`)
- Треугольник (`Triangle`) — задаётся тремя сторонами, с проверкой неравенства треугольника
- Квадрат ('Square')
Все фигуры имеют проверку на валидность
